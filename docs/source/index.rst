Welcome to Caligraphy's documentation!
======================================

**Caligraphy** (/kəˈlɪɡɹəfi/) is a hybrid programming language for scripting in the
modern age. It combines Python and Bash syntax, allowing you to automate other processes
with the ease of a Bash script while also giving you the advantages of a modern
programming language like Python.

Check out the :doc:`usage` section for further information, including
how to :ref:`installa` the project.

.. note::

   This project is under active development.
